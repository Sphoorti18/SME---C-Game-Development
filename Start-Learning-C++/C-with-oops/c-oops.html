Variables

In C++, variables are containers for storing data values.

Syntax :

#include

using namespace std;

//Before you start, this '//' is called comments to let the non-programmers'

// know what the heck you are doing with the code!

int main() {

// Integer variable

int num = 10;

// Floating-point variable

float pi = 3.14;

// Character variable

char letter = 'A';

// String variable

string name = "John Doe";

// Boolean variable

bool sTrue = true;

// Output variables

cout << "num: " << num << endl;

cout << "pi: " << pi << endl;

cout << "letter: " << letter << endl;

cout << "name: " << name << endl;

cout << "isTrue: " << isTrue << endl;

return 0;} //EOF main program :)

Keywords

C++ has reserved keywords that have special meanings.

Syntax :

#include using namespace std; int main() { // int is a keyword and return is a keyword too! int myInt = 5; return 0; }

Functions

Functions are blocks of code that perform a specific task.

Syntax :

#include using namespace std; int main() { // int is a keyword int myInt = 5; // if is a keyword if (myInt == 5) { cout << "myInt is equal to 5" << endl; } // class is a keyword class MyClass { // Class definition }; // return is a keyword return 0; } #include using namespace std; // Function declaration void greet() { cout << "Hello, world!" << endl; } int main() { // Function call greet(); return 0; //you can also remove this coz this was used by old folks }
